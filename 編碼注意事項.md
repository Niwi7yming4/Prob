# 編碼注意事項

本專案包含繁體中文與數學公式字串，所有原始碼請一律使用 UTF-8 without BOM。

## PowerShell 讀取中文檔

在繁體中文 Windows 上，舊版 PowerShell 可能用 Big5/CP950 讀取文字，造成中文變亂碼。讀取檔案時請指定 UTF-8：

```powershell
Get-Content -Raw -Encoding UTF8 .\APP.tsx
```

## 禁止的做法

- 不要用未指定 encoding 的 `Get-Content`、`Set-Content`、`Out-File` 處理中文檔。
- 不要把看到的 LaTeX 原始碼誤判成編碼錯誤；如果頁面顯示 `$H_0$` 或 `\theta`，通常是前端公式解析沒有吃到，不是 UTF-8 壞掉。

## 公式顯示

前端使用 KaTeX。內容中：

- 行內公式使用 `$...$`
- 獨立公式使用 `$$...$$`

修改後請執行：

```bash
npm run build
```

如果 build 通過但畫面仍顯示 LaTeX 原始碼，請檢查 `APP.tsx` 的內容解析器是否支援該段格式。
